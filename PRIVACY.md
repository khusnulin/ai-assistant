# Privacy Policy for Lightweight AI Assistant

**Last updated:** July 2026

This Privacy Policy describes how the "Lightweight AI Assistant" plugin for JetBrains IDEs handles your data.

### 1. Data Collection
This plugin **does not** collect, store, log, or share any of your personal data, source code, or prompts. We do not have any backend servers that receive your data.

### 2. Data Transmission
When you use the AI features of this plugin (e.g., chat, code completion), the selected code snippet and your prompt are sent **directly** from your IDE to the AI provider endpoint you have configured in the plugin settings (e.g., Ollama, OpenAI, Groq, Qwen, DeepSeek). 

### 3. Your Responsibility
You are responsible for the AI provider you choose. The plugin merely acts as a client to forward your requests to the API of your choice using your own API keys or local instance. Please review the privacy policies of the respective AI providers you use.
